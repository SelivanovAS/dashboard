⚖ Настройка дашборда

Подключите Google-таблицу с данными дел

  1. Создайте Google-таблицу со столбцами:
    Номер дела Дата поступления Истец Ответчик Категория Суд 1 инстанции Роль банка Статус Последнее событие Дата события Акт опубликован Результат Ссылка Заметки Апеллянт Дата публикации акта
  2. Откройте Файл → Поделиться → Опубликовать в интернете
  3. Выберите нужный лист, формат CSV, нажмите «Опубликовать»
  4. Скопируйте полученную ссылку и вставьте ниже

Или загрузите демо-данные чтобы посмотреть дашборд

Эта страница загружает данные напрямую из вашей Google-таблицы. Данные не передаются третьим лицам.
